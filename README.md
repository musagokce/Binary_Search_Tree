# Binary_Search_Tree
Write the Binary-Search-Tree stages of the array
[7, 5, 1, 8, 3, 6, 0, 9, 4, 2]

            7
         /    \
       5         8
     /    \          \
   1       6          9
  /   \
0     3
     /   \
   2      4

Big-O notation;
Average Case : O(logn)
Worst Case : O(n)
Insertion : O(logn)
